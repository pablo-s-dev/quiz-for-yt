## Detailed Description

```
Transform your YouTube learning experience with Quiz for YouTube! 🎓

Quiz for YouTube uses AI to automatically generate interactive quizzes from any YouTube video with captions. Test your understanding, improve retention, and make learning more engaging.

✨ KEY FEATURES

🤖 AI-Powered Quiz Generation
• Automatically creates multiple-choice questions from video transcripts
• Smart question generation based on video content
• 4 alternatives per question for comprehensive testing

🎯 Dual Generation Modes
• OpenAI API: Use your own API key for fast, reliable generation
• ChatGPT Mode: Free option using your ChatGPT account

⌨️ Full Keyboard Control
• Arrow keys (←/→) to navigate between questions
• Arrow keys (↑/↓) to select alternatives
• Space to select, Enter to submit
• Numbers (1-4) or letters (A-D) for quick selection
• Ctrl + Shift + Q to create/close quiz
• Escape to exit

📊 Progress Tracking
• Visual progress bar showing completion
• Question status indicators (correct/incorrect/unanswered)
• Real-time feedback on answers
• Review correct answers after submission

📚 Quiz History & Statistics
• Save quizzes for each video
• Track your performance over time
• View accuracy percentage
• See how many times you've generated quizzes

🌐 Bilingual Support
• Full English translation
• Complete Portuguese (PT-BR) translation
• Automatic language detection
• Manual language selection

🎨 Beautiful Interface
• Clean, modern design
• Smooth animations
• Responsive layout
• Intuitive navigation

🔒 Privacy First
• All data stored locally
• No tracking or analytics
• API keys encrypted locally
• Open source code

1. Open any YouTube video with captions
2. Click "Create Quiz" button below the video
3. Wait for AI to generate questions
4. Answer and test your knowledge!

Or use the keyboard shortcut: Ctrl + Shift + Q

• Fallback mechanisms for reliability
```

## Privacy Practices

### Data Usage

**Collects:**
- Quiz history (stored locally)
- User settings (stored locally)
- Video transcripts (temporarily, for quiz generation)

**Does NOT collect:**
- Personal identification
- Browsing history
- Analytics or tracking data

**Data Handling:**
- All data stored locally in browser
- API keys encrypted locally
- No remote servers or databases
- Data never shared with third parties

### Permissions Justification

**storage**
- Required to save quiz history and user settings locally

**activeTab**
- Required to inject quiz interface on YouTube pages

**host_permissions (youtube.com)**
- Required to access video transcripts and inject UI

**host_permissions (openai.com, chatgpt.com)**
- Required for AI quiz generation
